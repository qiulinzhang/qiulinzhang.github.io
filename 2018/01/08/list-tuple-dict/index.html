    list+tuple+dict

    Python
    创建时间:2018-01-08 15:45

    2. 二、tuple(元组)
    3. 三、dict(字典)

    一、list(列表)

    1. 声明一个列表:listName=[initial values],list使用[],可以包括不同数据类型，多个值用逗号隔开
    2. 访问索引从0开始，也可以利用负数从最后面开始，-1代表最后一个，-2代表倒数第二个
    3. 把一个列表或列表的一部分分配给一个变量，如下：
    • 如上，c=a[2:4]表示把a中索引为2到4-1的元素分配给c,符号2:4称为切片，无论何时我们在使用Python中的切片符号，它总是包括开始索引的元素，但不包括结束索引的元素
    • 切片符号包括第三个数字，叫做步长，即c=a[1:5:2],表示从索引位置起到最后5-1止每2个元素取一个，即c=[2,'4']
    • 切片符号的默认第一个数字是0，默认第二个数字的长度是列表的长度。
    1. 删除列表中的元素：del listName[index],添加数组尾元素：listName.append(value)

      二、tuple(元组)

    2. tuople的声明：tupleName=(initial value),tuple使用(),多个值用逗号隔开
    3. 一旦初始化，无法修改其值

      三、dict(字典)

    4. 声明：dictName={key1:value1,key2:value1},要求关键字唯一，要求使用{}
    • Note that you can use only immutable objects(like strings) for the keys of a dictionary but you can use either immutable or mutable objects for the values of the dictionary.
    • Remember that key-value pairs in adictionary are not ordered in any manner.
    • The dictionaries that you will be using are instance/objects of the dictclass
    • 使用dict()方法声明：nameAndAge=dict(Tom=10,Jack=20,Rose='Old')
    1. 往字典中添加元素，dictName[newKey]=newValue
      删除元素：del dictName[key]
    • The items method of the dictionary returns a list of tuples where each tuple contains a pair of items-the key followed by the value.
    1. Sequence
    • The slicing operation is used by specifying the name of the sequence followed by an optional pair of numbers separated by a colon within square brackets. Remember the numbers are optional but the colon isn’t.
    • Note that the slice returned starts at the start position and will end just before the end position.The start position is included but the end position is excluded from sequence slice.
    • You can also provide a third argument for the slice,which is the step for the slicing(by default,the step size is 1) [start:end:step]
