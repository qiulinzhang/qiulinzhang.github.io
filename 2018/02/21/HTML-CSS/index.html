二、CSS: Cascading Style Sheet 层叠/级联样式表

2.1 是一种格式设置规则，用于控制web页面的外观
2.2 如何让一个标签具有样式

  • 第一步，必须保证引入方式正确
  • 第二步，必须让CSS和HTML元素产生关联,也就是说得先找到这个元素
  • 第三步，用相应的CSS属性去修改HTML元素的样式
    2.3 如何去使用CSS
  • 在HTML网页中，加入一个style标签，在这个style标签里面写CSS代码
  • 可以直接把style里面的代码放到一个单独的css文件中，再通过link标签
  • 直接在HTML开始标签的style属性里面去写CSS代码
    2.4 三种基础选择器
  • id选择器，对应#:每个id的名称不能重复，一般应用于网页中有唯一的元素需要修饰
  • class类选择器，对应.:类名称可以出现多次多个元素可以为相同的类，一般应用于有多个元素具有同样的样式时
